## Description / 项目简介

为 AI 代理提供基于 Sui 区块链和 Walrus 分布式存储的持久化内存系统。系统采用三层存储模型：本地缓存（快速访问）+ Sui（审计日志）+ Walrus（数据存储）。支持多种记忆类型（对话、知识、状态、目标、工具结果），提供 REST API 接口和框架无关的设计。
