## Description / 项目简介

<!-- 一段话描述你的项目：做什么，为什么重要 / What it does, why it matters -->

Tidecast 是 DeepBook Predict（Sui 链上 BTC 预测市场）的桌面交易台：实时 SVI 波动率报价（devInspect 免 gas）、一键链上 mint/redeem（本地密钥进程内签名，无需钱包插件）、实时波动率微笑曲线与市场持仓热力图。已完成 testnet 真实集成，mint 交易 digest：`8wDTPzWhoq9YKVT95nVKJnBjqmMkPF1UNKCxvgiZmUif`。

Tidecast is a desktop trading console for DeepBook Predict, Sui's on-chain BTC prediction market: live SVI quotes via gas-free devInspect, one-click on-chain mint/redeem signed in-process by a local keypair (no wallet extension), plus a live vol smile and a market positioning heatmap. Real testnet integration — proof mint digest: `8wDTPzWhoq9YKVT95nVKJnBjqmMkPF1UNKCxvgiZmUif`.
